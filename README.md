# Проект: Место


![Место](src/images/Mesto.png)

Функционал:
* Редактирование профиля
* Добавление, удаление, лайк карточек
* Просмотр карточки в увеличенном размере

Работа выполнена с использованием:
* HTML
* CSS
* БЭМ
* Адаптивная верстка
* JavaScript
* Работа с Figma
* Работа с Perfect Pixel
* Подключение шрифтов
* PopUp
* Валидация форм
* Кнопки
